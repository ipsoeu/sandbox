Sentiment analysis about Contact tracing apps

The following analysis was performed using around 600K tweets collected from July 2020 to July 2021. In order to extract opinions and sentiments several pretrained BERT models have been used for each main langueges (English, Italian, French, German and Spanish). We denote around 250k tweets that not contain any sentiments or opinions and around 2% of unclassified tweets. The tweets with opinions are around three-quarters negative (253K )and one-quarter positive (78k).

The stacked barchar represents the tweets collected about the european contact tracing apps and classified as neutral, positive and negative.

Detected languages

For each language we used a pretrained BERT model where the involved training set was regarding social media and reviews, following the details:
  • English - model trained on ~58M tweets
  • French - model trained with 100k positive tweets and 100k negative tweets
  • German - the training data contains texts dorm different domains like Twitter, facebook and reviews
  • Italian - model trained with 45k tweets
  • Spanish - model trained with ~5k tweets
