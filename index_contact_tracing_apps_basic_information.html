Basic information on Covid-19 mobile contact tracing apps from tweets

An online search of the Twitter social media platform has been performed since July, 2020 to June 2021. To collect the tweets we used the official streaming Twitter APIs, in particular we open a streaming of tweets for each contract tracing apps. The table below shows the analyzed contact tracing apps and the search-keys used on the streaming.

Keywords used for streaming on Twitter about European contact tracing apps

For every individual tweets, a set of predeterminated analysis were collected in a real-time way combining the Tweepy library, Apache kafka and Elasticsearch. We developed a system able process different streaming of tweets and collecting place names, opinions, hashtags, entities and a quite number of aggregations.

Relevant information extracted from the collected tweets

The table below shows the number of tweets collected for each mobile contact tracing apps sorted by number of tweets. Moreover, the table reports the percentage of tweets with opinions, geo information and the number of detected EMM news.

The opinions were extraced respecting the five main languages detected in the collocetion which they are: English, French, German, Spanish and Italian. For the detection of the languages we used the standard engine evaluation language identification of Twitter.

Tweets that contained any kind of geo information were identified using the geoparser library Mordecai. The table reports the percentage of tweets which contains a place name in the tweet text or in the user profile.

Finally, a cross checking was performed between the dedicated EMM channel for Contact tracing apps and the tweets-linked pages in order to detect meaningful tweets.

Sentiment analysis

We extracted the opinions for each contact tracing apps from the tweet-text. The engine classifier is able to extract opinion from the English, French, German, Spanish and Italian languages. Futhermore, we can have one or more languages for a mobile contact tracing app, for example the Immuni app can have Italian and English languages.
