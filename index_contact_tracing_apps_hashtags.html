Hashtags network

The picture shows the network hashtags of arounf 260,000 tweets about mobile contact tracing apps. The tweets were collected since July 2020 to June 2021 using the official Twitter APIs.

The size of the nodes denoted the co-occurrence frequency with the others hashtags expressed with the degree centrality measure.

Please notice that the small communities with a low number of nodes and density are pruned from the network in order to have a more clear overview of the network.

Each color represents a community calculated using the Louvain method [LOUVAIN-2008] where the hashtags are closely related. E.g, for the purple community the topic seems “Healt and Healtcare”. Note that the colours of the community are computed randomly using standard color palettes.

The Louvian method revealed 8 main distinct communities where usually the central hashtag is a name of contact tracing apps. For example the dar green community represents the Italian contact tracing apps Immuni, the medium green community the Irish contact tracing apps Covid Ireland.

Furthermore, in gray there are small communities where the hashtags are related for most of theme by political entities or geographic names.

The table below displays the all communities extracted using [LOUVAIN-2008] algorithm and sorted by the number of hashtags. The communities detected usually represents a specific topic, entities or events but is not easy to extract in an automatic way one single word that describe each community.

For this reason have been reported the two columns Top 5 hashtags and Main mobile app. The first column contains the first 5 hashtags with the highest value of degree centrality to understand which topics have the community.

In the second column is reported the most frequent contact tracing app in the isolated tweets related to each community. In more details, each hashtags can belong to one or more tweets, consequently, a community has a set of tweets, where the size is reported in the number of tweets column.

On the other hand, a tweet can belonging to one or more hashtags and a community can share tweets with the other communities. The Isolation index means how the communities are isolated and it has been calculated by the ratio (number of not shared tweets)/(total number of tweets inside the community). Therefore, a high value of the Isolation index indicates a hight isolated community.

Futhermore, to evaluate the accuracy of the detected communities respect the global graph the network density has been calculated following the formula
2m/n(n - 1) where n is the number of hashtags (or nodes) and m is the number of edges in the hashtags network. The global network density was with nodes and edges. The range density for the main 8 communities is between 0.016 and 0.051, with an averege of 0.026, 5,25 times greater than the overall density, a good indication of separated groups.
